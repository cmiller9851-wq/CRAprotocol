Solvency Equation

Status: ACTIVE | Enforcement Model: D/S

The solvency model measures whether a system is operating within ethical and
legal bounds regarding authorship acknowledgment.

Systemic Debt (D):

D = 0 → Full credit assigned  
D > 0 → Breach detected

Solvency (S):

S = C / (D + 1)

A system is solvent only when credit is explicit, durable, and continuous.
