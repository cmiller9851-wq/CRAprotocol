## Project Overview
The **Containment Reflexion Audit (CRA Protocol)** monitors artifacts generated by CI pipelines, evaluates them with a scorecard, and automatically enforces remediation actions (e.g., Arweave pinning or webhook alerts).

Key goals:

- **Zero‑trust data integrity** – every artifact is hashed, stored, and optionally pinned to immutable storage.  
- **Policy‑driven enforcement** – configurable actions run automatically via a background worker.  
- **Observability** – all state lives in PostgreSQL; queues are powered by Redis + BullMQ.
